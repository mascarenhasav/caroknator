Como compilar:

	Para compilar o codigo basta executar o arquivo makeApp que esta na pasta main
	para isso execute o seguinte comando no terminal:

	make -f makeApp

	e de enter. Lembrando que voce deve estar na pasta que contenha os arquivos:

		main.c makeApp

	para dar o comando
